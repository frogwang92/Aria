# Aria
C++ based backtest platform

- C++ and Python mix programming with boost::Python.
- Datafeed and Algorithm can be implement in Python.
- Support OHLCV data.
- A simple GUI based on amcharts.
- Work on linux, build with CMake.
- CTP integrated.
- talib integrated.
